In school and after school STEM classes in Miami, FL. For grades k-12. Math, Coding, Minecraft, Robotics, 3D Design & Printing.

Employee dashboard with content management functionality.

Built with modern web technologies including:

- Next.js 14 with App Router
- Tailwind CSS for styling
- TypeScript
- Auth0 for Authentication
- PostgreSQL database
- Googleapis for content
